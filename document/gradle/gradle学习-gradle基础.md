## gradle安装



下载地址：https://gradle.org/releases/

```
// 进入到目录
$ cd ~/Downloads

// 查看文件
$ ls
$ ls ~/gradle

// 打开文件夹
$ open ~/gradle

// 解压到 ~/gradle 目录下
$ unzip gradle-6.8.3-bin.zip -d ~/gradle

// 清空控制台内容
$ clear

// 查看当前环境变量有哪些
$ echo $PATH

// 配置环境变量
$ export PATH=$PATH:~/gradle/gradle-6.8.3ex

// 检验配置是否成功
$ gradle -v
```
